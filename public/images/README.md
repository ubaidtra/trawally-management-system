# Images Directory

Place your company logo and other images here.

## Recommended Images:

- `logo.png` - Company logo (recommended size: 200x60px)
- `favicon.ico` - Browser favicon
- `hero-bg.jpg` - Hero section background (optional)

## Image Guidelines:

- Use optimized images for web
- Recommended formats: PNG for logos, JPG for photos
- Keep file sizes under 500KB
- Use descriptive filenames
